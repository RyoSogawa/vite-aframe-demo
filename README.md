# vite-aframe-demo #
This is demo AR projects with

- Vite
- TypeScript
- aframe
- GoogleMapAPI
- GiphyAPI
- Vercel Serverless Functions 

## Getting Started
install dependencies
```
yarn install
```

setting `.env` file.
```
GOOGLE_MAP_API_KEY=<YOUR_API_KEY>
GIPHY_API_KEY=<YOUR_API_KEY>
```

run on vercel.
```
yarn global add vercel
vercel dev
```
